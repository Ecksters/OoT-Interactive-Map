Ocarina of Time Version Differences

A comparison of actor differences between versions 1.0, 1.1, and 1.2 of The Legend of Zelda, Ocarina of Time.

Scope

This analysis does not cover all changes made between versions, simply the changes made to actor data within the ROM. As such, changes such as certain glitches no longer working, or grave holes no longer causing Link to grab or jump, will not be covered, there are other sites that cover those changes more in-depth, however, the goal of this project is to focus on the changes made to actors throughout the game between versions. The hope is to leave no doubt as to the exact changes made from version to version, and how extensive these changes were.
